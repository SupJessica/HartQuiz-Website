HartQuiz - Coming Soon

HartQuiz (formerly EZQuizTaker) allows you to easily build and administer or take multiple choice quizzes. Create quizzes with up to 3000 questions and up to 6 answers per question. View, print and/or email quiz results. Great for businesses, schools/educational institutes, and for entertainment!

About Hartquiz

With HartQuiz You Can... Create quizzes with up to 3000 questions
Build self-contained executable quiz packages
Enter e-mail information to have EZQuizTaker client automatically e-mail you the users results via SMTP
Attach 1-3 images per question with a thumbnail view that can be clicked to enlarge to full size
Password protect your Quiz Packages to prevent unauthorized editing or viewing of the answers
Password protect your Quiz Packages to prevent unauthorized users from taking your quizzes
Randomize the order of questions that are presented to the quiz taker as well as specify how many questions out of the total to display (e.g. randomly display 10 questions out of 100 that are set in the quiz project)
Import (and export) questions from CSV files
Set a limit on how many times a user can complete a quiz
Set a time limit of 1-100 minutes for the user to finish your quiz
Set a required passing score between 1-100%
Print test results or save results as HTML or Text

And much more!
